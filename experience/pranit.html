Interview Experience @ IBM

IBM - Software Developer

Round 1: Resume Shortlisting
Candidates were shortlisted based on resume evaluation, focusing on relevant projects, technical skills, and academic performance.

Round 2: Coding Round (Online)

  • Platform: HackerRank
  • Duration: 45 minutes
  • Programming Language Options: Java or C++
  • Key Topics: Arrays and Strings
  • Difficulty Level: Easy to Moderate

Round 3: Coding Round (Offline)

  • Two coding questions similar to the online round
  • Difficulty Level: Easy to Moderate

Round 4: English Assessment (Offline)

  • Included a short essay on a given topic
  • Grammar-based MCQs

Round 5: Group Discussion (Offline)

  • Topic-based group debate
  • Focused on communication skills

Round 6: Technical + HR Interview (Offline)

  • Technical Questions: Project discussion, Sorting algorithm implementation, SQL queries, OOPs in Java
  • HR Questions: Behavioral and situational questions assessing communication, attitude, and cultural fit

Result: Selected at IBM

This experience was shared by Pranit Mane, Computer Engineering student.
