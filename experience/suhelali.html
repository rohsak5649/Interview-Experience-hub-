Interview Experience @ Juspay

🎯 Juspay – Associate Software Development Engineer (React)

📞 Reaching Out & Getting In:
I proactively messaged Juspay employees via LinkedIn and email. After consistent follow-ups, I received a surprise call from HR for a screening round — testing my communication, passion, and intent for the SDE role.

🧪 Round 1: React-Based Product Challenge

  • Task: Build a Scratch-like animation system
  • Features included:
    • Sprite control
    • Animation switching
    • Collision detection
    • Visual block system
  • Tech Used: ReactJS, dynamic sprite handling, animation logic
  • Outcome: Cleared the round with a call from HR saying “Let’s move to the next one!”

💻 Round 2: DSA Hackathon (Part A & B)

  • Part A: "Tree of Space" Problem
  • Focus: Clarity, problem-solving approach, design thinking
  • Part B: Code Optimization + Thread Safety
  • Tasks: Optimize Part A, ensure thread safety, implement custom data structures
  • Duration: 5–6 hours per part (mentally intensive)

🧠 Round 3: Technical Interview

  • Panel: Senior SDE-3 Engineer
  • Topics Covered:
    • Self-introduction and resume/project discussion
    • CS Fundamentals: OOPs, DBMS, JavaScript internals, System Design basics
    • React Deep Dive:
      • Debug and explain useState & useEffect behavior
      • Implement useState and useEffect from scratch in plain JavaScript
  • Feedback: “Good attempt, but you need to go deeper into how things actually work.”

☎️ The Call:
A week later, I received a call I thought was a rejection. Instead, she said:
“Congratulations! When can you join?” 🥳

✅ Final Result: Selected at Juspay

Role: Associate Software Development Engineer (React)

Status: Accepted
