Interview Experience @ Synechron Technologies Pvt. Limited

Synechron Technologies Pvt. Limited - Software Developer

Round 1: Online Assessment
This round included:

  • Basic aptitude questions
  • Easy-level coding problems
  • Grammar-based questions
It tested logical reasoning, verbal ability, and programming fundamentals.

Round 2: Coding Assessment
Conducted at the company premises. Candidates were divided based on preferred language – Java or Python.
Topics covered:

  • Arrays
  • Strings
  • HashMap
  • Bit Manipulation
  • Stack and Queues
There were two coding questions of easy to medium difficulty.

Round 3: Technical Interviews
Two technical rounds (~15–20 minutes each), covering:

  • Core and Advanced Java
  • OOPs concepts
  • Final Year Project discussion
  • Basic introduction and resume-based questions

Round 4: HR Interview
Duration: ~30 minutes. Topics included:

  • Complete resume discussion
  • Relocation preferences
  • General behavioral questions

Round 5: Topic Writing
Candidates were given four topics to choose from and had to write on one. This was a non-elimination round to assess written communication skills.

Result: Selected at Synechron Technologies Pvt. Limited
