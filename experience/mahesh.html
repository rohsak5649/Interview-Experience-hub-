Interview Rounds Overview

Round 1: Resume Shortlisting & Screening

  • CGPA Criteria: Above 8.5
  • Skills: Full Stack Development, DSA
  • HR Screening Call with basic questions

Round 2: Online DSA Coding Test

  • Platform: GeeksforGeeks
  • 3 DSA Problems (2 Medium, 1 Hard)
  • Cleared: Solved 2 fully + passed test cases in 3rd

Round 3: Technical Interview (1 hr 15 min)

  • OS: Kernel, types of OS, deadlocks
  • CN: Protocols, IPv4/IPv6, tunneling
  • DBMS: Queries, Normalization, Transactions
  • Frontend: React + CSS Flexbox/Grid component
  • DSA: Medium 2D Matrix problem (solved)

Round 4: Managerial Interview (1 hr)

  • Backend: API methods (GET, POST, etc.), Express.js + DB
  • React: Components, hooks, lifecycle
  • OOP: 4 pillars, overloading vs overriding
  • DSA: Array-based question (solved)
  • Behavioral: Decision-making, leadership

Round 5: HR Interview (~20 mins)

  • Family Background
  • Career goals and 5-year plan
  • Availability and expectations
